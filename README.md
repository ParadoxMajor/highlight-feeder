# Highlight-Feeder

Just install it, set your category, and any other desired settings, and you're good to go! Highlighted posts will be automatically crossposted to [r/highlights](https://www.reddit.com/r/highlights/)!

Any issues, feedback, or feature requests, please send a [modmail to r/MajorParadoxApps](https://www.reddit.com/message/compose?to=%2Fr%2FMajorParadoxApps&subject=Highlight-Feeder%20feedback).

Note: You can view your current highlighting status using the ... overflow menu on your main subreddit page. It will show if you have it enabled in your settings and will give you a status, such as if the app is currently paused, disabled, or an updated is required to continue the highlighting.
